Covid Database API (W.I.P.)

Below are some of the api endpoints available. Please use them responsibly!

POST /api/login

  • For users authorization.
  • Please provide user login credentials in the form of json: {"username": USERNAME, "password": PASSWORD}

GET /api/data/countries

  • Gets countries with Covid data available.
  • Data is protected. Please provide Authorization header with the token you got from login.

GET /api/data/countries/<country>

  • Gets detailed Covid data for a given country.
  • Data is protected. Please provide Authorization header with the token you got from login.

GET /api/users

  • Gets a list of user account names.
  • Data is protected. Please provide Authorization header with the token you got from login.
  • Admin access only.

GET /api/users/<userid>

  • Gets detailed User data for a user.
  • Data is protected. Please provide Authorization header with the token you got from login.
  • Admin access only.
